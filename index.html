03 Proyectos

Trabajo en producción.

2024 — presente Plataforma · PWA
nowdii.com

Nowdii

El directorio de tu ciudad. Perfiles reales, activos y cerca de ti.

El problema

Los negocios locales en ciudades medianas y pequeñas de México carecen de un canal de visibilidad digital confiable. Las plataformas existentes son genéricas y no reflejan la realidad del comercio local.

La solución

Una PWA con autenticación multi-proveedor, perfiles verificados, sistema de insignias, búsqueda por zona, reseñas, historias y un panel de administración completo. Diseñada desde Zacualtipán para México.

HTML / CSS / JS Supabase PostgreSQL Vercel PWA OAuth Resend
2023 — presente Sitio comercial
deedpri.com

deedpri

Centro de impresión y diseño. Calidad profesional, atención personalizada.

El problema

Un centro de impresión físico necesitaba presencia digital seria para captar clientes antes de la visita. La competencia no tenía sitios optimizados ni comunicación clara de servicios.

La solución

Sitio con sistema de design tokens (60+ variables CSS), página fotográfica con estética cinematográfica, calculadora de cotización y PageSpeed de 99/84. Ahora es canal activo de clientes.

HTML / CSS / JS Design Tokens Responsive SEO PageSpeed 99

05 Proceso

Cómo trabajo.

  1. 01

    Descubrimiento

    Entiendo el problema antes de proponer solución. Conversamos sobre contexto, usuarios, objetivos de negocio y restricciones técnicas para definir qué estamos construyendo y por qué.

  2. 02

    Diseño

    Establezco la arquitectura de la información, el flujo del usuario y el sistema visual. Prototipado rápido para validar decisiones antes de escribir una línea de código.

  3. 03

    Desarrollo

    Implementación iterativa con entregas frecuentes. Código limpio, componentes reutilizables y rendimiento como requisito, no como lujo. Pruebas en cada paso antes de avanzar.

  4. 04

    Lanzamiento

    Despliegue en producción, monitoreo, documentación clara y transferencia de conocimiento. El trabajo no termina al subir el código — termina cuando el producto funciona y tú lo entiendes.
